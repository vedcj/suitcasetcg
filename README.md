# Suitcase TCG
## version 0.10.2

# Features :
- Check prices from the ff. sources:
  - Yuyu-tei
  - Big web
  - TCG Corner
- Price conversion to local price.
- Batch search for card prices.
- Search for English name and display it.
- View Card Info from search results.
- Full card image display.
